# User Documentation

## Deskripsi
Dokumen ini berisi panduan penggunaan aplikasi.

## Cara Menggunakan

1. Buka halaman login
2. Klik tombol register
3. Isi data diri
4. Klik create account
5. Masukkan kode verifikasi (1234)
6. Login menggunakan akun
7. Masuk ke dashboard

## Kesimpulan
Aplikasi mudah digunakan dan user friendly.

# Software Test Plan

## Deskripsi
Dokumen ini berisi rencana dan hasil pengujian sistem.

## Pengujian yang Dilakukan
- Login berhasil
- Login gagal
- Registrasi berhasil
- Verifikasi kode
- Penyimpanan data

## Metode
Pengujian dilakukan secara manual dengan mencoba setiap fitur.
